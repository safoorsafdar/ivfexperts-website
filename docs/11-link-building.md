## Executive Summary

The site has excellent on-page SEO and growing content — but backlinks are the single most powerful ranking factor Google uses, and the site currently has minimal external authority. A medical site in a competitive space (IVF, fertility) needs authoritative backlinks from medical directories, health publications, and topically relevant sites. The good news: Pakistan's fertility space has very few sites actively building links, so even a modest link building effort produces outsized results. Focus on quality over quantity — 10 links from HFEA, Healthline, and Pakistan medical directories outweigh 500 links from random directories.

**Biggest opportunity:** Getting cited in authoritative fertility resources (HFEA, ASRM-cited content) and Pakistan health publications positions Dr. Adnan as the go-to expert and drives both rankings and AI citations simultaneously.

### ✅ Task 6: HARO / Connectively expert sourcing
- **Tool:** Connectively (formerly HARO — Help a Reporter Out) at connectively.us
- **How it works:** Journalists post requests for expert sources; you respond with Dr. Adnan's quotes
- **Fertility-relevant queries to respond to:**
  - IVF success rates
  - Male infertility
  - PCOS treatments
  - Fertility preservation
  - International fertility treatment
- **Process:**
  1. Create a free Connectively account
  2. Set up alerts for keywords: "fertility", "IVF", "infertility", "reproductive"
  3. Respond to 3–5 relevant queries per week with Dr. Adnan's expert quote
  4. Responses that are selected get published with a link to the expert's site
- **Note:** Response quality matters more than quantity. 2-3 paragraph expert quotes beat 1-sentence replies.
- **Priority:** 🟡 Medium (ongoing, low effort once set up)

## LINK MONITORING

### ✅ Task 10: Set up backlink monitoring
- **Free tools:**
  - **Google Search Console → Links report:** Shows which sites link to ivfexperts.pk — check monthly
  - **Ahrefs Webmaster Tools (free tier):** ahrefs.com/webmaster-tools — verify site, get basic backlink data
  - **Moz Link Explorer (free):** Limited free checks for backlink profile overview
- **What to monitor:**
  - New backlinks acquired (positive)
  - Lost backlinks (need to investigate — did a page 404?)
  - Toxic/spam links (disavow if necessary via Search Console Disavow tool)
  - Competitor backlinks (find where competitors get links — target same sources)
- **Monthly action:** Log new links in a tracking spreadsheet: Date | Source URL | Linking page | Target page | DA/Authority
- **Priority:** 🟡 Medium (set up once, check monthly)
